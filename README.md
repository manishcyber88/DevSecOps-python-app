# DevSecOps Python App 🚀

A secure Flask-based web application integrated with a DevSecOps CI/CD pipeline using GitHub Actions, Docker, automated testing, and security scanning.

---

## 🔥 Features

- User Authentication
- File Uploads
- Admin Dashboard
- Search Functionality
- CI/CD Pipeline
- Dockerized Deployment
- Automated Testing
- Security Scanning

---

## 🛠️ Tech Stack

- Python
- Flask
- GitHub Actions
- Docker
- HTML/CSS/JS
- SQLite

---

## 🐳 Docker Setup

### Build Docker Image

```bash
docker build -t devsecops-app .
```

### Run Docker Container

```bash
docker run -p 5000:5000 devsecops-app
```

---

## 🚀 Run Locally

```bash
pip install -r requirements.txt
python app.py
```
